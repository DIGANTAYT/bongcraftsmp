# Connecting your Next.js Store to Tebex (Headless API)

To use this premium web design with your active Tebex store, you should use the **Tebex Headless API**. This allows you to host this custom website on Vercel/Netlify for free while using Tebex to handle checkout payments and execute commands in-game.

---

## 1. Retrieve your Tebex Headless API Token
1. Log in to your **Tebex Creator Panel** (https://creator.tebex.io).
2. Navigate to **Integrations** -> **Headless API** in the sidebar.
3. Enable the Headless API if it isn't already enabled.
4. Copy your **Public API Token**.

---

## 2. Connect the Next.js Code to Tebex API
Instead of using our local cart state, you will call Tebex's endpoints. You can install the official Tebex Headless SDK or fetch endpoints directly:

### Fetching Packages Dynamically
Update your product loaders to fetch your real categories and prices directly from Tebex:
```typescript
// Example fetch call inside your component
const response = await fetch("https://headless.tebex.io/api/v1/packages", {
  headers: {
    "X-Tebex-Token": "YOUR_PUBLIC_HEADLESS_TOKEN"
  }
});
const data = await response.json();
```

### Creating a Tebex Cart & Adding Items
When a player enters their Minecraft username and adds an item, you create a cart on Tebex:
```typescript
// 1. Create a Cart
const cartResponse = await fetch("https://headless.tebex.io/api/v1/carts", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "X-Tebex-Token": "YOUR_PUBLIC_HEADLESS_TOKEN"
  },
  body: JSON.stringify({
    username: minecraftUsername, // e.g. Neeeeeel
    game_type: "minecraft"
  })
});
const cart = await cartResponse.json();

// 2. Add Package to Cart
await fetch(`https://headless.tebex.io/api/v1/carts/${cart.id}/packages`, {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "X-Tebex-Token": "YOUR_PUBLIC_HEADLESS_TOKEN"
  },
  body: JSON.stringify({
    package_id: 123456, // Your Tebex Package ID
    quantity: 1
  })
});
```

### Launching Checkout
Instead of a simulated modal, redirect the player directly to the secure Tebex Web checkout url generated by the cart endpoint:
```typescript
// Redirect the player to Tebex to complete payment
window.location.href = cart.links.checkout;
```
Once the payment is completed, Tebex handles the payment and automatically fires the rank commands (e.g. `/lp user {name} parent add supreme`) on your server.

## 3. Deploying the Storefront (Free & Fast)
The easiest way to put this website online:
1. Push your workspace folder to a private **GitHub Repository**.
2. Go to **Vercel** (https://vercel.com) and link your GitHub account.
3. Import the repository and click **Deploy**.
4. Vercel will build your Next.js application and give you a free production URL (e.g. `bongcraft-store.vercel.app`), which you can then link to your domain `store.bongcraft.in`.
